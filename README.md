# Focus Grid Extension

A grid-based Pomodoro extension for Chrome.

## Objective
To calibrate actual daily focus capacity by observing the quantity and distribution of grid blocks, mitigating overly idealistic planning and expectations.

## Features
**Numberless Tracking:** Eliminates the countdown timer to prevent time-induced anxiety and recalibrate the subjective perception of time.

## GUI Controls
From left to right on the main panel:
1. **Duration Input:** Enter the target time in minutes.
2. **Start:** Initiate the timer.
3. **Rest:** Manually insert a rest block.
4. **Reset Timer:** Cancel the current ongoing timer.
5. **Clear Grid:** Reset all recorded blocks.
6. **Floating Widget (📌):** Summon the anchor widget to the current page (Disabled on `chrome://` URLs).
7. **Theme Toggle:** Switch between Light and Dark mode.

**State Evaluation:** 
Click on completed blocks to toggle between **Green** and **Brown**. Use these colors to label qualitative states based on personal habits (e.g., Green for effective focus, Brown for rest or internal friction).

## Installation
1. Open your browser and navigate to `chrome://extensions/`.
2. Enable **Developer mode** (usually in the top right corner).
3. Click **Load unpacked** (top left) and select the folder containing all the project files.
